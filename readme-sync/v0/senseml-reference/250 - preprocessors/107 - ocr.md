Use this preprocessor to selectively OCR pages.

Parameters
====

| key                       | value        | description |
| ------------------------- | ------------ | ----------- |
| `type` (**required**)     | `ocr`        |             |
| match (**required**)      | Match object |             |
| pageOffset (**required**) | number       |             |

Examples
====

```json
{
  "preprocessors": [
    {
      "type": "ocr",
      "match": { "type": "first" },
      "pageOffset": 0
    },
    {
      "type": "ocr",
      "match": { "type": "first" },
      "pageOffset": 3
    }
  ],
  "fields": [
    {
      "id": "policy_number",
      "anchor": {
        "match": {
          "type": "equals",
          "text": "policy number:"
        }
      },
      "method": {
        "id": "label",
        "position": "right"
      }
    }
      ]
}
```
