Solution: Mix it up
---


After you've completed all the Mix it up steps in the tutorial, the final config is:

```
{
  "fields": [
    {
      "id": "second_most_popular_snack",
      "anchor": "second",
      "type": "string",
      "method": {
        "id": "row",
        "position": "right",
        "tiebreaker": "last"
      }
    },
    {
      "id": "snack_table",
      "anchor": "tables and rows",
      "type": "table",
      "method": {
        "id": "fixedTable",
        "columnCount": 6,
        "columns": [
          {
            "id": "third_column",
            "index": 2
          },
          {
            "id": "sixth_column",
            "index": 5,
            "type": "phoneNumber"
          }
        ],
        "stop": {
          "type": "startsWith",
          "text": "mix it up"
        }
      }
    }
  ]
}
```
